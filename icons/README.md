# @mts-ds/icons

Пакет web-иконок Дизайн-Системы МТС.

Состав пакета:

- `/interface/*` -- Иконки со страницы `Interface Icons`. Иконки в формате SVG;
- `/interface-documents/*` -- Иконки со страницы `Interface Icons Documents`. Иконки в формате SVG;
- `/web2/*` -- Устаревшее. Копия `/interface/*`.

Каждая иконка -- это набор вариантов начертания расположенных в индивидуальной папке.
Название папки равно названию иконки.
Название файла варианта иконки собирается из аттрибутов варианта.
Например, у иконки есть атрибуты `size` равный `16` и `style` имеет значение `outline`, то имя файла будет: `size-16-style-outline.svg`.

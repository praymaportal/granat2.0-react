# granat2.0-react

Корпоративная библиотека компонентов на React 18 + Vite. Архитектура и структура каталогов ориентированы на PrimeReact, но компоненты реализуются с нуля.

## Состав
- `button/`, `spinner/` — реализованные компоненты
- `resources/` — базовые стили и токены (шрифты, темы, типографика)
- `icons/` — набор иконок MTS
- `dev/` — кастомный сторибук (меню слева, отображается только выбранный компонент)

## Запуск dev
```bash
npm install --cache ./.npm-cache
npm run dev
```

## Сборка библиотеки
```bash
npm run build
```

## Storybook
Dev‑сторибук показывает варианты компонентов. Каждый компонент добавляется в `dev/stories/`.
